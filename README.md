# Bus Tracking System (Tier-2 / Tier-3 Cities)

This project is a real-time bus tracking application with a simple frontend and a backend powered by Firebase.

# Features

Live bus location tracking on map
Real-time updates using Firebase
Simple and responsive frontend interface
Easy integration with GPS or driver mobile app

# Tech Stack

Frontend: HTML, CSS, JavaScript
Backend: Firebase (Realtime Database / Firestore)

# How It Works

Bus location data is sent to Firebase.
The frontend fetches the latest coordinates.
Bus position is updated on the map in real time.

# Purpose

This project demonstrates a basic implementation of a real-time vehicle tracking system that can be extended for public transport tracking.

# Future Improvements
